# フルスクラッチバ美肉プロジェクト

## 概要

諸々のツールをなるべく使わずにリアルタイムで美少女になるためのプロジェクト。
本プロジェクトの一部は、[CubismSDKおよびサンプルモデル](https://github.com/Live2D/CubismNativeSamples)を利用してており、
`resouces`、`cubism`および`src`ディレクトリ以下は、Cubism SDK Release Licenseに従います。
